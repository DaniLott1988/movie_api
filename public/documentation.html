Movie It Documentation

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres.

Endpoints

  • URL: '/'
  • Request: Start Page
  • Description: Starting Page, Welcoming page.
  • URL: '/login'
  • Request: For authentication process
  • Description: For Users to log in in order to be able to access their profile, see movies, add to their favorites, or make changes to their profiles.
  • URL: /movies
  • Request: GET list of all the movies
  • Description: JSON that has the movies data as objects.
  • URL: /movies/:Title
  • Request: GET movie by title
  • Description: JSON object retrieved through title's data.
  • URL: /directors
  • Request: GET list of all the directors
  • Description: JSON that has the directors data as objects.
  • URL: /directors/:Name
  • Request: GET director by name
  • Description: JSON object retrieved through name's data.
  • URL: /genres
  • Request: GET list of all the genres
  • Description: JSON that has the genres data as objects.
  • URL: /genres/:Name
  • Request: GET genre by type
  • Description: JSON object retrieved through type's data.
  • URL: /users
  • Request: GET list of all the users
  • Description: JSON that has the users data as objects.
  • URL: /users/:Username
  • Request: GET user by username
  • Description: JSON object retrieved through username's data.
  • URL: /users
  • Request: POST new user
  • Description: Confirmation message is show as new user is added.
  • URL: /users/:Username/movies/:MovieID
  • Request: POST new favorite to the list of the user
  • Description: Confirmation message is show as new favorite is added to the list.
  • URL: /users/:Username
  • Request: PUT (update) username
  • Description: Confirmation message is show as selected username is updated.
  • URL: /users/:Username
  • Request: DELETE user through Username
  • Description: Confirmation message is show as user is deleted.
  • URL: /users/:Username/movies/:MovieID
  • Request: DELETE a movie from favorites of the user
  • Description: Confirmation message is show as movie excluded from favorites.
